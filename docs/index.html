How to Setup?

1. For setting up Spell Me first download the compressed file from Home Page by clicking        Download      .


2. After downloading, On Chrome click on the three dots ( ) on top. Then in tools section select Extensions or paste this chrome://extension URL in the action bar .

3. On the page enable developer options from the top right corner. Then three new buttons should appear in the toolbar. The first option will be “Pack Extension”. Click on it, then choose the compressed file you have downloaded.
